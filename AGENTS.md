## Project Summary
A high-fidelity clone of the Soonix waitlist landing page (Option C). The project features a minimalist, tech-forward aesthetic with a hero section for email signups, product video previews, a FAQ accordion, and a floating promotional widget.

## Tech Stack
- Framework: Next.js (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Animation: tw-animate-css, Lucide React (icons)
- Deployment: Vercel

## Architecture
- `src/app/`: Core application logic and routing.
- `src/components/sections/`: Modular frontend sections (Navbar, Hero, FAQ, etc.).
- `src/app/globals.css`: Global styles and Tailwind theme configurations.

## User Preferences
- Clean, minimalist UI design.
- Pixel-perfect cloning from provided Framer URL.
- No comments unless requested.

## Project Guidelines
- Use "use client" for components requiring state or interactivity.
- Follow the established design tokens in `globals.css`.
- Ensure mobile responsiveness for all sections.

## Common Patterns
- Section-based component structure.
- Responsive design using Tailwind utilities.
- Lucide React for consistent iconography.
